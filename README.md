
## Projeto Experts Club

<p>Esse projeto foi desenvolvido durante uma aula do Paulo Salvatore no Experts Club da Rocketseat.Consiste em um CRUD feito em memória com NestJS, com o objetivo de conhecer o Nest.</p>
 

## Instalação

```bash
$ npm install

ou

$ yarn add
```

## Executando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testando

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
